
# Battleship
An implementation of the classic Battleship game built with WPF using the MVVM pattern.
This project was created as a school project and focuses on clean architecture, UI interaction, and the separation of game logic.

## ▶️ Getting Started

### 🔽 Option 1: Download
**Requirements** <br>
Windows OS

 1. Download the latest [Release](https://github.com/Tutenik/Battleship/releases/latest)
 
 2. Run the application .exe file



### 💻 Option 2: Run from Source
**Requirements** <br>
Windows OS <br>
 .NET 9 or newer <br> 
Visual Studio 2022 or newer
>⚠️ In Visual Studio 2022, you need to have enabled [.slnx support](https://community.sw.siemens.com/s/article/How-to-open-slnx-solution-files-in-Visual-Studio-2022)

<br>

1. Clone the repository
```
git clone https://github.com/Tutenik/Battleship.git
```
2. Open the solution
```
 Battleship.slnx
```
3. Build and run the project in **Visual Studio**.

## 🧪 Project Highlights
Separation of UI and logic using MVVM <br>
Custom ship sets using .json files
